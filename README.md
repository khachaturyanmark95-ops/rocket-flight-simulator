# Rocket-flight-simulator
A Python simulation of rocket flight with physics calculations and graphs

## What it does
-Simulates rocket launch with thrust, mass, gravity and fuel
-Calculates acceleration, speed and height over time
Generates graphs showing flight data

## Physics used 
-Newton's Second Law: F = ma
-Acceleration = (Thrust / Mass) - Gravity

## Technologies
-Python 3
-Matplotlib
